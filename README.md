## 2) CNN 대상 Adversarial Attack에 대한 방어 PoC


Author: Tackhyun Jung

Status: 진행 중

### 핵심목표
1) OpenCV의 다양한 기술을 통해 여러 종류의 스캔 이미지나 영수증을 보기 좋게 가공해줌
2) REST API 기반의 웹 서비스 수행 예정

---

### 사용된 기술
* OTSU-Thresholding
* Morph erode and open
* Find Contours
* Perspective Transform

---

### Requirement
* Python 3.6
* cv2
* imutils
* numpy

---

### Usage

```
$ Python Main.py
```
